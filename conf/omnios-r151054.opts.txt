the above information | enter

# Post-install: "Once back at the main menu, ... rpool now has a working
# and mounted boot environment, per above". Stays on screen for the full
# install duration (several minutes), then waits indefinitely for enter.
back at the main menu | enter

# Installer main menu (Welcome / Shell / Reboot / Halt). The enter selects
# the default item, which on a finished install is Reboot -- the VM powers
# off the install media and boots from disk into first-boot init.
